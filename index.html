Predicting Early-lactation Diseases in Cows with Milk FTIR

This study aims to predict subsequent postpartum diseases at dairy cows using FTIR spectra collected from daily milking procedures, with a long goal of supporting large-scale health monitoring program in modern dairy farms. Specifically, we devise 'combined framework', a mechanism that enables high-speed batch pre-processing spectra and tailor ensemble machine learning algorithms for diseases prediction. Using this framework, we successfully predict the overall health condition, metritis and clinical mastitis on one commercial dairy farm in upstate NY.
